サマログ！クエスト

【開き方】
index.html をダブルクリックすると使えます。
インターネット接続やインストールは不要です。

【基本の流れ】
1. 右上の「保護者」を開く
2. お子さまを追加し、子どもごとに夏休み全体の課題を登録
3. 1アイテムに必要な合計量と、1日の獲得上限を設定
4. 必要に応じてChatGPT用プロンプトで課題一覧をJSON化して一括登録
5. 保護者カレンダーに、全員または特定の子どもの予定を登録
6. 「キッズ」画面で子どもを選び、できた課題と量を記録
7. 教科をまたいだ合計が目標に達するとアイテムを獲得
8. アイテムでモンスターを倒すと、次のモンスターが自動出現
9. 夏休み全体の課題がすべて終わると黄金の宝箱を獲得

【毎日かならずクエスト】
保護者画面で「花の水やり」「お手伝い」などを登録できます。
ポイントとは別の毎日チェックとして、子ども画面に表示されます。

【モンスターバトル】
敵ごとに弱点アイテムが1種類あります。
「今回必要なアイテム」に表示されたアイテムで攻撃してください。
ポイント達成時には、現在の敵に必要なアイテムを獲得できます。

ポイント条件と、その日の「毎日かならずクエスト」を達成すると、
今日のクエスト欄に「アイテムをゲット！」ボタンが現れます。
このボタンを押した時点で、アイテムが所持品に入り使用可能になります。

【冒険図鑑】
子ども画面の「冒険図鑑をひらく」から確認できます。
未入手のアイテムと未撃破の敵は影で表示されます。
すべて解放すると、図鑑の最後に秘密の宝箱が現れます。

【ふりがな・音】
画面上部の「ふりがな ON/OFF」で、主な漢字の読みを切り替えられます。
「音 ON/OFF」で、アイテム獲得・攻撃・撃破・敵出現などの効果音を切り替えられます。
設定はブラウザに保存されます。

【進捗ポイント】
1ページ、1日分、1回、1枚をそれぞれ1ポイントとして合算します。
例：国語3ページ＋算数2ページ＝5ポイントです。

【複数人管理】
画面上部の名前タブで子どもを切り替えます。
課題、進捗、アイテム、モンスター戦績は子どもごとに保存されます。

【保存について】
データは、このHTMLを開いたブラウザ内に保存されます。
ブラウザのサイトデータを消すと記録も消えるためご注意ください。

【PINについて】
保護者画面に4桁PINを設定できます。
同じ端末・ブラウザ内での簡易ロックです。
